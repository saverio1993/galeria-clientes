¿Cómo creo el token?
  1. Ve a github.com/settings/tokens?type=beta
  2. Generate new token → Fine-grained
  3. En Repository access elige Only select repositories → saverio1993/galeria-clientes
  4. En Permissions → Contents → Read and write
  5. Generate → copia el token (empieza con github_pat_)

Con un token fine-grained solo este repo puede ser modificado, aunque alguien obtenga la URL del admin.

¿Y si quiero subir archivos .ARW (Sony RAW)?

El navegador no puede convertir .ARW directamente. Para fotos RAW usa el script local:

  1. Copia los .ARW a la carpeta images-origen/ del proyecto
  2. Ejecuta subir.bat (doble click) — convierte + sube a GitHub automáticamente
  3. O manualmente: py -3 scripts/convertir.py + git push
